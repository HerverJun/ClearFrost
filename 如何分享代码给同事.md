# 如何与同事分享代码

本文档将指导你如何让同事访问并学习 **清霜视觉 (ClearFrost)** 项目的代码。

---

## 🔗 仓库地址

你的代码已托管在 **Gitee**（码云）上：

> **https://gitee.com/jiao-xiake/ClearForst**

---

## 📖 方式一：直接分享链接（最简单）

把上面的仓库地址发给同事，他们可以：

1. **在线浏览代码**：直接在网页上查看所有文件和提交历史
2. **克隆到本地**：使用以下命令将代码下载到他们的电脑

```bash
git clone https://gitee.com/jiao-xiake/ClearForst.git
```

---

## 🔒 如果仓库是私有的

如果你的仓库设置为 **私有 (Private)**，同事需要被邀请才能访问：

### 邀请步骤：

1. 登录 Gitee，进入仓库页面
2. 点击 **「管理」** → **「仓库成员管理」**
3. 点击 **「添加仓库成员」**
4. 输入同事的 **Gitee 用户名或邮箱**
5. 选择权限：
   - **观察者**：只能查看代码（推荐用于学习）
   - **开发者**：可以提交代码
   - **管理员**：完全权限
6. 点击 **「确定」** 完成邀请

同事会收到邀请通知，接受后即可访问仓库。

---

## 📦 方式二：下载 ZIP 压缩包

如果同事不熟悉 Git，可以直接下载压缩包：

1. 打开仓库页面：https://gitee.com/jiao-xiake/ClearForst
2. 点击 **「克隆/下载」** 按钮（绿色按钮）
3. 选择 **「下载 ZIP」**

这样他们就能获得一个包含所有源代码的压缩包。

---

## 🛠️ 同事拿到代码后如何运行

### 环境要求

- **操作系统**：Windows 10/11 (x64)
- **开发工具**：Visual Studio 2022（或 VS Code + .NET SDK）
- **.NET 版本**：.NET 8.0 SDK

### 编译运行步骤

1. 用 Visual Studio 打开 `ClearFrost/ClearFrost.csproj`
2. 选择 **Debug | x64** 配置
3. 按 **F5** 运行项目

或者使用命令行：

```bash
cd ClearFrost
dotnet build -c Debug -p:Platform=x64
dotnet run
```

## 💡 学习建议

建议同事按以下顺序阅读代码：

1. **入口点**：`Program.cs` - 了解程序启动流程
2. **检测服务**：`Services/DetectionService.cs` - 核心检测逻辑
3. **多模型管理**：`Core/MultiModelManager.cs` - 多模型自动切换
4. **主界面逻辑**：`Views/主窗口.*.cs` - 界面交互
5. **前端界面**：`html/index.html` + `html/js/` - Web UI 实现

## ❓ 常见问题

### Q: 同事说无法访问仓库？
A: 检查仓库是否为私有。如果是，需要在「仓库成员管理」中邀请他们。

### Q: 编译时报错缺少 DLL？
A: 确保 `ClearFrost_Lite/` 目录下有必要的 SDK 文件（如 MVSDK.dll）。

### Q: 如何查看历史版本？
A: 在 Gitee 仓库页面点击「提交」可以查看所有历史提交记录。
